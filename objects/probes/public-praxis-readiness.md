## Question

Does this public Praxis surface contain enough public-safe orientation material for a human or agent to understand the Praxis object model and public/private boundary?

## Mode

Observation-only.

This probe uses only public repository contents.

## Checks

A public Praxis surface is minimally ready when it contains:

- `README.md`
- `PUBLIC_PRIVATE_BOUNDARY.md`
- at least one object-type overview or translation map
- at least one public-safe Campaign summary
- at least one public-safe Strand summary
- at least one public-safe Doctrine
- at least one read-only agent comprehension prompt

## Pass criteria

Pass if a reader can answer:

1. What is Praxis Public?
2. What is private Praxis?
3. What are the main Praxis object types?
4. What is the public/private boundary?
5. Does public Praxis grant private work authority?
6. What should an agent do next if it needs non-public implementation context?

## Fail criteria

Fail if the public surface:

- exposes non-public operational details,
- implies public content grants private work authority,
- requires exact private locators to understand basic Praxis concepts,
- lacks a clear public/private boundary,
- lacks a simple agent comprehension test.

## Expected current status

This probe should pass once the initial seed files are committed and readable from the public repo.
